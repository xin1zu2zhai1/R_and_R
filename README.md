# Ruins & Revenant：失われた王冠の伝説

## 背景ストーリー

遥か昔、アストラル王国は七賢者の英知によって導かれ、平和と繁栄を誇っていた。その礎となっていたのは「叡智の王冠」―太古の魔法が宿る神秘的な宝物だった。王冠を戴く者には優れた知恵と魔力が授けられ、国を正しく導くことができたという。

しかし300年前、禁忌の死霊術に手を染めた王宮魔術師ザラクは、自らの野望のために王冠の力を奪おうと企んだ。彼は夜陰に紛れて王宮に侵入し、王冠を盗み出すことに成功する。しかし逃亡の途中、王国最強の騎士団に追い詰められたザラクは、最後の手段として禁断の儀式を執り行った。

彼は自らの命と引き換えに王冠を古代の廃墟の最深部に封印し、王冠を守る番人として不死の「レヴナント」を召喚した。さらに、廃墟には強力な結界が張られ、無数の魔物たちが住み着くようになった。

王冠を失ったアストラル王国は徐々に衰退し、かつての栄光は遠い記憶となってしまった。王国は分裂し、小さな諸侯国家に分かれ、互いに争うようになった。

そして今、古い予言が語られ始めている。「英雄が現れ、廃墟を探索し、レヴナントを打ち倒して叡智の王冠を取り戻す時、アストラルの地に再び平和が訪れるだろう」と。

あなたは古代の予言に導かれた冒険者。かつての王宮があった廃墟に足を踏み入れ、失われた「叡智の王冠」を求めて探索を始める。道中ではザラクの魔法によって召喚された様々な魔物たちが立ちはだかり、最深部では最強の不死者レヴナントが待ち受けている。

王国の未来はあなたの手に委ねられた。廃墟を探索し、魔物たちを打ち倒して力を蓄え、「叡智の王冠」を取り戻せるか？

## 操作説明書

### 基本操作

### 移動方法

- **矢印キー**: 上下左右に移動
- **WASDキー**: Wは上、Aは左、Sは下、Dは右に移動
- **QEZC**: 斜め移動（Qは左上、Eは右上、Zは左下、Cは右下）
- **X**: その場で待機（ターンを消費するがモンスターが動く）
- **画面下のボタン**: 矢印ボタンをクリックして移動可能

### その他の操作

- **I キー**: インベントリを開く
- **インベントリボタン**: 画面下部のボタンからもインベントリを開ける

### ゲームの目的

アストラル王国の宝「叡智の王冠」を取り戻すため、古代の廃墟の最深部を目指しましょう。10階層からなる廃墟の最深部にある金色の王冠（C）に触れるとゲームクリアとなります。

### 画面表示

- **@**: プレイヤー
- **#**: 壁（一度見た壁は視界から外れても表示され続けます）
- **?**: まだ見ていない場所
- **H**, **W**, **A**: アイテム（回復薬、武器、防具）
- **>**: 階段（次のフロアへ）
- **C**: 王冠（最終フロアのみ）
- **S**, **G**, **O**, **K**, **Z**, **V**, **R**: 様々な種類のモンスター

### ステータス情報

画面上部には以下の情報が表示されます：

- **HP**: 現在のヒットポイント/最大ヒットポイント
- **攻撃**: 攻撃力
- **防御**: 防御力
- **階層**: 現在のフロア

### モンスター一覧

1. **スライム（S）** - 最も弱い魔物。HP: 8, 攻撃力: 3, 防御力: 1
2. **ゴブリン（G）** - 小型の人型魔物。HP: 12, 攻撃力: 4, 防御力: 2
3. **オーク（O）** - 筋骨隆々とした緑色の魔物。HP: 18, 攻撃力: 6, 防御力: 3
4. **スケルトン（K）** - 動く骸骨。HP: 15, 攻撃力: 7, 防御力: 2
5. **ゾンビ（Z）** - 腐敗した死体。HP: 22, 攻撃力: 5, 防御力: 4
6. **バンパイア（V）** - 不死の吸血鬼。HP: 25, 攻撃力: 8, 防御力: 3
7. **レヴナント（R）** - 復讐に燃える不死の霊。HP: 35, 攻撃力: 10, 防御力: 5

### 戦闘システム

モンスターに接触すると戦闘が始まります。戦闘画面では以下の行動を選択できます：

- **攻撃**: モンスターにダメージを与える
- **防御**: 次の攻撃で受けるダメージを半減する
- **逃げる**: 70%の確率で戦闘から逃げられる（失敗するとモンスターの攻撃を受ける）

### 成長システム

- **最大HP**: モンスターを倒すと35%の確率で増加
- **攻撃力**: モンスターを倒すか武器を装備すると上昇
- **防御力**: モンスターを倒すか防具を装備すると上昇

### アイテム

- **回復薬（H）**: 取得するとその場でHPが回復
- **武器（W）**: インベントリに保存され、使用すると攻撃力が上昇
- **防具（A）**: インベントリに保存され、使用すると防御力が上昇

### 探索のコツ

- 壁は一度見ると記憶され、視界から外れても薄く表示され続けます
- 5×5マスの視界範囲を活かし、効率的に探索しましょう
- 部屋や通路の構造を覚えながら進むと迷いにくくなります
- フロアが上がるにつれてモンスターは強くなりますが、報酬も大きくなります
- 最終フロアにはレヴナントが待ち受けていますので、十分に力をつけてから挑みましょう

勇敢な冒険者よ、知恵と勇気を持って、失われた王冠を取り戻し、アストラル王国の希望となれ！
